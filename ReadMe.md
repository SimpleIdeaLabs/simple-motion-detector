# Simple Motion Detector
Motion Detector with Python

## Description

A simple implementation of open cv python used for video processing and detecting movements

## Getting Started

### Dependencies

* opencv-python
* simpleaudio

### Installing

* pipenv install

### Executing program

* pipenv shell
* python main.py
